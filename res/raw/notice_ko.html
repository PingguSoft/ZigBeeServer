 블루투스 도달 거리는 핸드폰과 블루투스 UART 모듈에 의해 결정됩니다

 이 어플리케이션은 통신 끊김으로 인한 피해에 대해 책임이 없습니다

 사용자의 주의가 필요합니다
•
 24시간 무료 평가입니다. 계속 사용하시려면 구매버튼을 눌러주세요


버전 1.1
• 통신 테스트 기능 추가

• 블루투스나 지그비 리시버를 사용시 RTL(Round Trip Time)을 측정할 수 있습니다
• 배터리 설정 수정

• 배터리 설정을 선택된 배터리 셀 단위 전압으로 설정하도록 수정
• u-blox AssisNow Online 지원

• GPS FIX를 빠르게 도와주는 정보를 서버로 부터 받아서 GPS 모듈에 전송하는 기능입니다

• u-blox GPS 모듈 Neo-6M 이상의 버전에서 동작합니다

• 정보 전송후 수십초 내에 GPS FIX가 됩니다

• 자세한 정보는 해당 기능의 설명을 참고하세요
• 스테이션 모드 수정

• 비행 경로 아이콘 추가

• 고도 정보 오류 수정
• 조종기 및 스테이션 화면 메인메뉴에서 선택


버전 1.0.1
• 인터보드 BT9M, BT10 블루투스 수신기 지원

• 일반 수신기와 마찬가지로 MSP를 사용하지 않아 자세, 고도, 방향, 배터리 게이지를 사용할수 없습니다


버전 1.0.0
• 핸드폰의 블루투스를 이용하여 MultiWii 기반의 기체를 조종할 수 있습니다
• 더 먼 도달거리를 위해 ZigBee module (ProBee-ZE20S)을 지원합니다
• RC 튜닝, 쓰로틀 설정, 배터리 설정과 AUX 설정을 할 수 있습니다
• 조종기 모드와 스테이션 (모니터링) 모드가 지원됩니다



• 조종기 모드

• 자세, 고도, 방향, 배터리 게이지를 제공합니다

• 4개의 AUX 스위치를 사용할 수 있습니다

• 가속도 센서와 지자기 센서 교정이 지원됩니다

• 트림 조정과 쓰로틀 홀드가 지원됩니다

• 핸드폰의 가속도 센서를 이용하여 Roll(Aileron)과 Pitch(Elevator)를 조종합니다

• 가속도 센서의 민감도를 조종할 수 있습니다

• 화면상의 스틱을 이용하여 Throttle과 Yaw(Rudder)를 조종합니다



• 스테이션(모니터링) 모드

• 자세, 고도, 방향, 배터리 게이지를 제공합니다

• 4개의 AUX 스위치를 모니터링 할 수 있습니다

• 가속도 센서와 지자기 센서 교정이 지원됩니다

• GPS 연결 상태를 확인 할 수 있습니다

• 구글맵과 다음맵이 지원됩니다

• 지도상에 이동 경로를 표시합니다

• 이동 경로를 KML파일로 저장합니다 (/sdcard/BTCon4MultiWii)

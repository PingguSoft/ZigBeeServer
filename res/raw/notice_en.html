Version 1.1
• Com Test feature added

• RTL(Round Trip Time) measurement feature is added if bluetooth or zigbee receiver is used
• Battery setting modified

• Based on voltage of selected battery cell
• u-blox AssisNow Online feature added

• This feature is sending data from server to GPS module for fast GPS FIX

• Working with u-blox GPS modules equal or higher version than Neo-6M

• GPS will be fixed in short time after sending data to GPS module

• Refer to the description in that function for detailed information
• Station mode modification

• Add icon to flight path

• Fix altitude bug
• Controller and Station mode selection in main menu for easy mode changing


Version 1.0.1
• Interboard bluetooth receiver BT9M, BT10 supported

• Attitude, Altitude, Heading and Fuel gauge are not supported because they are same with other RC receivers not using MSP


Version 1.0.0
• Control copters based on MultiWii with your phone using bluetooth
• ZigBee module (ProBee-ZE20S) supported for long range
• MultiWii configs such as RC tuning, Throttle settings, Battery scale and AUX settings are supported
• Controller mode and station(monitoring) mode are supported



• Controller mode

• Attitude, Altitude, Heading and Fuel gauge are supported

• 4 AUX switches are supported (Low/High)

• ACC and MAG calibrations are supported

• Trim and throttle hold are supported

• Control Roll(Aileron) and Pitch(Elevator) using acceleration sensor on your phone

• Sensitivity of Acceleration sensor is adjustable

• Control Throttle and Yaw(Rudder) using OSD stick



• Station(Monitoring) mode

• Attitude, Altitude, Heading and Fuel gauge are supported

• Monitoring 4 AUX switches (Low/Mid/High)

• ACC and MAG calibrations are supported

• Monitoring GPS connection state

• Google map and Daum map(For Korean users) are supported

• Marking the flight path on the map

• Saving flight path to KML file for Google map (/sdcard/BTCon4MultiWii)
